# homebrew-tap

## What is a Tap?

Taps are third-party repositories for Homebrew. They allow users to install packages which are not merged into [homebrew-core](https://github.com/Homebrew/homebrew-core).

You can learn more about Taps [here](https://docs.brew.sh/Tap).

## Usage

To add this tap as a repository to Homebrew, run the following command:
```
brew tap FlabbergastAgency/tap
```

## Tools

Below is a list of tools available from this tap, along with install commands for each.

| Tool                                                         | Command                                         |
|--------------------------------------------------------------|-------------------------------------------------|
| [create-api](https://github.com/FlabbergastAgency/CreateAPI) | `brew install FlabbergastAgency/tap/create-api` |
